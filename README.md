# KANBAN-BOARD

It's a desktop app. 
https://kanban-board-ashen.vercel.app/

*__Stack__: React, CSS, JS, Redux, Firebase, DND-kit*

## Already made functionality:

+ adding Columns;
+ editing column's header;
+ drag and drop Columns;
+ adding Tasks;
+ editing Tasks;
+ drag and drop Tasks;
+ delete Tasks and Columns;
